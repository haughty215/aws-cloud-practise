Compute

Run applications with EC2, serverless with Lambda, and manage scaling with Auto Scaling groups.

EC2 Instances

Virtual machines with flexible CPU, memory, and storage options.

aws ec2 run-instances --image-id ami-1234 --instance-type t3.micro
Auto Scaling

Scale capacity automatically based on demand.

aws autoscaling create-auto-scaling-group --auto-scaling-group-name demo --min-size 1 --max-size 3
Lambda

Run code without provisioning servers.

aws lambda create-function --function-name hello --runtime nodejs18.x --role arn:aws:iam::123456789012:role/service-role/role-name --handler index.handler
