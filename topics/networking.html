Networking

Core building blocks for connecting resources in AWS: VPCs, subnets, route tables, internet/NAT gateways, and more.

VPC Basics

A Virtual Private Cloud is your isolated network within AWS.

aws ec2 create-vpc --cidr-block 10.0.0.0/16
Subnets

Subnets partition your VPC into smaller networks across AZs.

aws ec2 create-subnet --vpc-id vpc-1234 --cidr-block 10.0.1.0/24
Route Tables

Define where traffic should go. Associate with subnets.

aws ec2 create-route-table --vpc-id vpc-1234
Internet Gateway vs NAT Gateway

IGW allows inbound/outbound internet for public subnets; NAT allows outbound-only for private subnets.
