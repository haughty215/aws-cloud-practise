Lab: S3 Static Website

Create an S3 bucket and enable static website hosting.

Step 1: Create Bucket

Bucket names are globally unique.

aws s3api create-bucket --bucket  --region  --create-bucket-configuration LocationConstraint=

Step 2: Upload Site

Upload your static files.

aws s3 sync . s3://

Step 3: Website Hosting

Enable static website hosting.

aws s3 website s3:/// --index-document index.html --error-document 404.html
Consider using CloudFront for HTTPS and global distribution.
